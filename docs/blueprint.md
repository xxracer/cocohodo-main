# **App Name**: Cocohodo Katy coffe shop

## Core Features:

- Hero Section: Displays a vintage, scholarly-themed introduction to Cocohodo Katy.
- Menu Display: Showcases menu items with elegant serif typography.
- Catering & Private Events Contact Form: Allows users to inquire about catering and private events through a sophisticated form.
- About Us Section: Presents the story of Cocohodo Katy with film-grain style photography.
- FAQ Accordion: Provides answers to frequently asked questions in a collapsible accordion format.
- Real-Time Business Hours: Displays current business hours in the website footer.
- Google Reviews Carousel: Fetches and displays recent Google Reviews in a carousel format to increase credibility.

## Style Guidelines:

- Primary color: Warm Amber (#D4A373) evoking a cozy, inviting feel, like the amber lighting in Agora Houston.
- Background color: Parchment (#F5F5DC) to emulate the texture of aged paper.
- Accent color: Dark Wood (#4A3A30) to reinforce the Classical Library aspect and provide contrast.
- Menu, About Us Sections font: 'Literata', a serif typeface for headers and body to capture the vintage scholarly aesthetic requested.
- Minimalist icons reflecting Korean design, used sparingly to guide user interaction without cluttering the layout.
- Use a professional grid layout that provides structure and balance while allowing the vintage and minimal elements to be presented cleanly.
- Subtle, smooth transitions on hover states and accordion dropdowns to enhance user experience without distracting from the core content.
